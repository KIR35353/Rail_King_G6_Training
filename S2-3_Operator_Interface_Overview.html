What to Verify by Model
Gauge Ranges
Numeric ranges and normal operating bands vary by model. Confirm acceptable values in the model manual before interpreting readings.
Warning Indicator Logic
Amber and red indicator trigger thresholds differ between RK22, RK23, and RK24. The same reading may be normal on one model and a warning on another.
Advanced Display (RK24)
The RK24 IFM CR1081 display provides additional diagnostic screens. Consult IFM documentation in C6 for menu navigation.
⚠ Caution: Do not use a visual indicator reading from one model as a baseline for a different model without checking model-specific thresholds first.

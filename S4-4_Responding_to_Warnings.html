Responding to Warnings
Section 4
1
Value Enters Upper or Lower Caution Zone
Gauge reading approaches band limit but no lamp illuminated. Operator reduces load or duty cycle proactively. Record observation in shift log.
REDUCE LOAD
log observation
2
Amber Warning Lamp Illuminates
Non-critical fault or threshold crossed. Reduce load immediately, identify the system. Note fault code. Continue to nearest safe stop — do not complete the full work cycle.
SAFE STOP
note fault code
3
At Safe Stop — Assess and Decide
Operator looks up fault code in model manual. Three outcomes: (a) operator-resolvable — act and restart. (b) monitor-and-continue — per manual guidance. (c) escalate — call service.
ASSESS
3 outcomes
4
Red Warning Lamp — Critical Fault
Critical system threshold crossed. Stop machine at next safe point within the current motion — do not continue a work cycle. Apply park brake, lower any raised load safely, shut down engine.
STOP NOW
lower load first
5
Machine Secured — Escalate to Service
Park brake set, engine off, key removed. Complete the fault report form with fault codes, conditions, and timeline. Machine is out-of-service until service team clears and restarts.
OUT OF SVC
service team
Amber vs. Red — Know the Difference
Amber = advisory — the system has crossed a threshold but has not failed. You have time to reach a safe stop and assess.

Red = critical — a protected system is at or beyond its safety boundary. Stop is required immediately, not at the next convenient point.
Lowering a Load Before Shutdown
When a red lamp illuminates while a load is suspended or raised: lower the load to the ground or a stable position before shutting down. Hydraulic pressure will bleed off after shutdown — an unsecured load will drop if left raised.
Do Not Clear and Retry
Resetting a fault lamp and restarting does not repair the fault. If the same fault reappears within one work cycle, the machine must be taken out of service. Two occurrences = service mandatory.
Warning response hierarchy: Proactive load reduction → amber safe stop + assess → red stop now + secure → escalate. Each level requires a log entry before resuming or handing off.

When to Use Model-Specific Manuals
Section 2
Reference selection rule: Start with the G6 Operator Reference Guide for general workflow. Switch to a model manual the moment you encounter thresholds, procedures, or components that may differ by model.
✓ Use G6 Reference Guide
STARTUP
General startup flow and safety gate sequence applies across the family.
SHUTDOWN
Controlled shutdown concepts and idle-before-shutdown requirements are shared.
HANDOFF
Operator handoff structure and documentation expectations are common.
SUPPORT
Escalation pathway and form-use procedures apply regardless of model.
SAFETY
Role boundaries and stop/escalate rules are universal across the Gen6 family.
⚠ Switch to Model Manual
LIMITS
Any specific PSI, temperature, RPM, or pressure value — verify against model manual.
FAULTS
Fault code lookup and diagnostic steps differ by model controller and sensor set.
SERVICE
Maintenance intervals, fluid specifications, and service torque values are model-specific.
DISPLAY
Menu navigation and display behaviour differ between RK22/23 and the RK24 IFM system.
OEM
Any subsystem with an OEM vendor component requires the C6 vendor documentation (RK24 only).
📄 Which Manual to Pick
RK22
Use RK22.pdf for all model-specific procedures, limits, and service guidance on RK22 units.
RK23
Use RK23.pdf for all model-specific content on RK23 units — do not substitute RK22 values.
RK24
Use RK24 Service Manual plus C5 drawings and C6 OEM docs as needed for RK24 units.
UNSURE
If you cannot confirm which manual applies, stop and escalate. Do not guess across models.
Decision shortcut: Numbers and component details → model manual. Workflow and process → G6 Reference Guide. Both uncertain → escalate before acting.

Shared Principles Apply
Blue subsystems follow common operating concepts across RK22, RK23, and RK24. G6 Operator Reference Guide covers these.
Model-Specific Validation Required
Orange subsystems are unique to RK24. Always consult RK24 Service Manual and C6 OEM docs for these systems.
Threshold Differences Exist
Even for shared subsystems, operating limits and threshold values differ by model. Never assume identical numbers.
Practical Rule
Identify the subsystem → check if it's shared or model-specific → select the appropriate reference document before acting.

When to Use These Documents
Controller Fault
Fault code appears on the CR1081 display that is not in the operator manual. Where do I look?
Doc 07b (CR1081 System Manual) for alarm page structure, then 07e (CR0020 Software Manual) for fault code definitions and the input/output condition that triggers each code.
Parameter Verification
After a controller replacement, the machine behaviour seems different. What checks are needed?
Doc 07f (CR0020 System Manual) — verify CAN bus addresses and I/O assignments. Doc 07e (Software Manual) — confirm all parameter values match the unit's build specification.
Camera System
The rear camera image is not appearing on the in-cab monitor. How do I diagnose this?
Start with Doc 07g (connection instructions) — verify cable continuity and connector seating. Then check Doc 12 (Safety Vision monitor) for input source selection. Doc 07i for camera power voltage spec.
CAN bus integration: The CR0020 controller, CR1081 display, and camera system share a CAN network. A wiring fault or address conflict on any node can cause symptoms across all three systems simultaneously. Always check the system manual (07f) before replacing components.
The CR0020 is the brain of the RK24 control system. All fault codes originate here — the display is the window, not the source.

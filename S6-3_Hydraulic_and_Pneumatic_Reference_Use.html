Hydraulic and Pneumatic Reference Use
Section 6
Hydraulic and Air System OEM Documents
REXROTH
Variable Piston Pumps and Directional Valves
Docs 11a–11f: A10V pump, A10VSO pump, D-Pilot valve, CD-7 valve, SM-300 service manual, relay valve
Primary hydraulic power source for travel and work circuits. Use pump docs for flow/pressure curves, case drain specs, and shaft torque limits. Use valve docs for spool identification, pressure settings, and pilot signal requirements.
MARZOCCHI
GHM2 Gear Pump
Doc 09: GHM2 Pump Catalog
Auxiliary gear pump for low-pressure circuits (steering, pilot supply, or cooling fan). Catalog covers displacement options, pressure ratings, and port configurations.
SUN HYD
Sun Hydraulics Cartridge Valves
Docs 15a–15b: Service bulletin and circuit drawing A71012VB03
Sun cartridge valves are used in manifold blocks throughout the RK24 hydraulic circuits. The circuit drawing is essential for identifying cartridge positions and setting sequences.
HYDAC
DF/LF Filter Assemblies
Doc 06: Hydac DF LF filter assembly
Return-line and pressure-line filtration. Use for element cross-reference, bypass valve cracking pressure, and visual indicator interpretation during maintenance.
BENDIX / AAA
Air Dryer and Brake Circuit Valves
Docs 03, 01a–01b: AD-9 air dryer, remote pilot and solenoid air valves
AD-9 covers purge cycle, heater element, and desiccant service intervals. Air valve spec sheets define port sizes, operating pressure ranges, and coil voltage for the brake and auxiliary pneumatic circuits.
When to Open These Documents
These are service-level references. The scenarios below show the specific question that sends a technician to each document bundle.
Pump / Flow Question
Hydraulic pressure is building but flow seems low under load. Where do I look?
Start with Rexroth 11a/11b (pump specs) — check variable displacement setting and case drain flow limits. If pump is within spec, move to 11d (CD-7 valve) for internal leakage data.
Valve / Circuit Question
A cylinder is not holding position at rest. What's the reference?
Sun Hydraulics 15b circuit drawing — identify the counterbalance or load-holding cartridge for that circuit. Use 15a service bulletin for leakage specs and test procedure.
Filter Question
Bypass indicator is showing on the return filter. Is the element serviceable?
Hydac Doc 06 — cross-reference element part number, confirm bypass cracking pressure, and check if the indicator is a flow indicator or differential pressure switch.
Air System Question
Air reservoirs are not building to full pressure and purge frequency has increased. Where do I start?
Bendix AD-9 Doc 03 — check purge cycle timing, desiccant condition, and heater element operation. If dryer is OK, check AAA solenoid valve (01b) for seat leakage.
Service-level boundary: Operators read gauges and report. Technicians open the OEM documents. Do not adjust hydraulic or pneumatic settings without the correct document in hand.

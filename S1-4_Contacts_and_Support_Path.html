Contacts and Support Path
Section 1
1
Observe and Document
Operator notes abnormal condition. Record time, symptoms, and operating state before any action.
2
Check Approved Procedure First
Consult operator reference and model manual. If a procedure addresses the condition, follow it.
3
Notify Supervisor
If procedure does not resolve or condition is unclear, notify lead or supervisor immediately.
4
Submit Equipment Support Request
Use form_equip_tech_support.pdf. Include unit ID, symptoms, times, and steps already taken.
5
Contact Technical Support Direct
Use contacts.pdf to identify the correct support team and communication channel. State urgency clearly.
What to Include in Every Request
Unit model and serial number
Date, time, and operating conditions at onset
Exact symptom description — no assumptions
Steps already taken and outcome
Current unit status (running, shut down, secured)
Escalation Speed Matters
Earlier escalation reduces diagnostic time
Complete data prevents call-backs and delays
Secure the unit if safe operating limits are unclear
